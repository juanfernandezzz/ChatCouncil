<h1 align="center">ChatCouncil</h1>

<div align="center">

Multi-model AI Chat Interface — compare responses from all major AI models side by side.

</div>

## Features

- 🤖 Chat with multiple AI models simultaneously (GPT-5.5, Claude, Gemini, Grok, DeepSeek, and more)
- 💬 Side-by-side comparison of model responses in a grid layout
- 🚀 Switch between grid layouts: 2, 3, 4, or 6 panels
- 🔑 BYOK (Bring Your Own Key) for API-based models
- 🎨 Dark mode with customizable theme colors
- 📚 Prompt Library with custom and community prompts
- 💾 Conversation history saved locally
- 📥 Export and Import all your data
- 🔗 Share conversations via markdown

## 🤖 Supported Models

- GPT-5.5 / GPT-5.5 Thinking
- Claude Sonnet 4.6
- Gemini 3.5 Flash / 3.1 Pro / 3 Flash
- Grok 4.3
- DeepSeek-V4 Pro
- Kimi K2.7 Code
- MiniMax M3
- GLM-5.2
- Qwen3.7 Plus
- Perplexity Sonar
- OpenRouter (BYOK)
